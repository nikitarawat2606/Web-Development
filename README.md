## How to Run :

1. Download or clone this repository to your local machine.
2. Navigate to the project folder where your HTML file is located.
3. Locate the main HTML file (e.g., `index.html`).
4. Double‑click the file, or right‑click → *Open With* → select your preferred browser (Chrome, Edge, Firefox, etc.).
5. The webpage will open in your browser and display the styled headings and paragraphs.

> 💡 Tip: If you’re using an IDE like NetBeans or VS Code, you can also right‑click the HTML file and choose **Run File** to launch it directly in your browser.
